# Liashkevich Code Studio 🚀

Портфоліо та студія розробки фронтенд-розробника **Andriy Liashkevich**, який спеціалізується на створенні **MVP**, **SaaS** та **PWA** додатків.

## ✨ Про студію

Ми створюємо высокофункціональні веб-додатки, які:
- 🚄 **Запускаються швидко** - мінімальний time-to-market
- 📱 **Виглядають як мобільні app** - класний UI/UX
- 💰 **Допомагають бізнесу заробляти** - з перших користувачів

### Послуги

- **MVP & SaaS розробка** - від ідеї до запуску
- **Вебсайти** - посадкові сторінки, портфоліо, корпоративні сайти
- **Інтеграції** - API, платіжні системи, аналітика
- **3D & Іграшки** - прототипи ігор, 3D візуалізація в Blender
- **Ціна** - в 2x дешевше, ніж середній ринок

## 🎯 Технологічний стек

- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Node.js, API інтеграції
- **Deploy**: Vercel, Azure

## 🚀 Запуск локально

### Вимоги
- Node.js версії 16+
- npm або yarn

### Встановлення

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/cheradey1/liashkevich-code-studio.git
   cd liashkevich-code-studio
   ```

2. Встановіть залежності:
   ```bash
   npm install
   ```

3. Запустіть dev-сервер:
   ```bash
   npm run dev
   ```

4. Відкрийте http://localhost:5173 у браузері
